# NLW Pocket Javascript

## Objetivo
Desenvolver uma aplicação fullstack (ReactJS + Nodejs) chamada in.orbit.
A aplicação tem como o objetivo ser um gerenciador de metas semanais.

## Tecnologias
### Frontend
- ReactJS
- TailwindCSS
- Lucide React
- Radix UI
- TanStack Query
- React Hook Form
- Zod
- Dayjs

### Backend
- NodeJS
- Fastify
- Fastify CORS
- Docker
- PostgreSQL
- Drizzle ORM
- Zod
- Dayjs
